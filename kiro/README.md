# DevLoop for Kiro

Copy `steering/devloop-requirements-to-stories.md` into your project's `.kiro/steering/`
and the `specs/_template/` files into `.kiro/specs/<feature>/` as starting points.

The steering file is set to `inclusion: manual` — reference it in chat with
`#devloop-requirements-to-stories` (or change the front-matter to `inclusion: always` to
keep it active in every session). It mirrors Kiro's native requirements → design → tasks
spec flow.
